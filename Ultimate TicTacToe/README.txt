Project Title: Ultimate TicTacToe

Description: A game that uses Python graphics to play a variation of 
"TicTacToe"

How to run: Run UltTicTacToe.py in the same folder as "graphics.py"

How to play: Rules for Ultimate TicTacToe here: 
https://www.thegamegal.com/2018/09/01/ultimate-tic-tac-toe/
Variations: If a board ties, it counts for neither player.
