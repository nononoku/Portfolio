Title: Minesweeper
Description: C++ implementation of the popular game Minesweeper
Programmed by Noah Kuperberg 2/9/2022

Usage: (in a terminal)
make minesweeper
./minesweeper e|m|h

e: easy (9x9 board, 10 mines)
m: medium (16x16 board, 40 mines)
h: hard (16x30 board, 99 mines)

To play, follow prompts in terminal. Normal Minesweeper rules apply.
